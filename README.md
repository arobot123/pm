# Project Manage

## 注意事项

- 无第三方担保，个人对个人的不接。双方都没保障，出问题可能撕破脸不了了之。
- 上来就直接说给我开发一个什么软件，不说具体需求的不接。无需求文档的不靠谱。
- 尽量442收费，开工收40%，中期收40%，项目结束收20%。开始不给钱的不接。
- 记得留个bug，防止不给尾款。
